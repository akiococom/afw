Required install operations:

	1. Create working directories and set permission to 777.
			xxx/logs
			xxx/tmp
			
	2. Setting ini files
			Copy .dist file to same directory and remove .dist extension and set your parameters in these files.
			xxxx/etc/Afw-customer-ini.php.dist
				-> Afw-customer-ini.php
			xxxx/etc/Afw-db-ini.php.dist
				-> Afw-db-ini.php
			
	3. Setting .htaccess file
			Copy .htaccess-dist to same directory and rename .htaccess and set your install directory in this file.
			xxx/www/.htaccess-dist
				-> .htaccess
